# Testownik

Aplikacja służy do nauki pytań testowych, które program pozyskuje z odpowiednio przygotowanych plików tekstowych.

## Przygotowanie pliku

Na początku należy przygotować odpowiednio pytania w dowolnym edytorze tekstowym

```
1. Pytanie
   a. odp 1
   b. odp 2
   c. odp 3
   d. odp 4

2. Pytanie 2
   a. odp 1
   b. odp 2
   c. odp 3
```
